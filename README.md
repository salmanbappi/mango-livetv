# Mango Live TV Extension

Aniyomi extension for Mango Live TV streams.

## Features
- **Direct Stream Parsing:** Fetches live channels directly from the Mango TV server.
- **Categories:** Supports multiple categories including Sports, Movies, News, etc.

## Installation
1. Install Aniyomi or Anikku.
2. Add this extension from the repository.
